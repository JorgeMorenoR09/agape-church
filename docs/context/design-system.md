# Design System

## Branding

Main phrase:

**"El amor hace la diferencia"**

Usage:
- Highlight in the **hero section** as the primary H1 or tagline
- Reinforce in **call-to-action sections**
- Reinforce in the **donation section**
- Visually impactful but minimalist presentation

## Typography

Headings

Cormorant Garamond

Body

Raleway

Accent Script

Allura

Usage guidelines

- Cormorant Garamond: H1, H2, H3 y títulos de sección
- Raleway: párrafos, navegación, formularios y textos de soporte
- Allura: lema de marca y frases destacadas en Hero, Donaciones y Footer
