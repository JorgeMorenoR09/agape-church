# Project Overview

This project is a modern church website.

The structure is inspired by the reference site:

https://ejedefe.com/

However the design must be unique.

The website must support two languages:

Spanish
English

Main features:

- church information
- weekly devotional
- ministries
- prayer requests
- donation section
- events

The website must prioritize:

performance
SEO
mobile-first design
accessibility

## Branding

Main phrase:

"El amor hace la diferencia"

This phrase must be highlighted in the hero section and reused across the site.

It represents the core message of the church.
