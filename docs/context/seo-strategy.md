# SEO Strategy

SEO optimization is a core requirement.

---

# Goals

- High ranking on search engines
- Fast page load
- Structured content

---

# Technical SEO

All pages must include:

Title tag
Meta description
Canonical URL
Open Graph tags
Twitter card tags

Example:

<meta name="description" content="Christian church focused on faith, community and spiritual growth.">

---

# Semantic HTML

Use semantic elements:

header
nav
main
section
article
footer

Example structure:

header
main
  section
  section
footer

---

# Heading hierarchy

Each page must have one H1.

Example:

H1 Page Title

H2 Section titles

H3 Subsections

---

# Image SEO

All images must include:

alt attributes
optimized size
lazy loading

---

# Performance

Target Lighthouse score:

90+

---

# URL Structure

Use clean URLs.

Examples:

/es/devocional
/en/devotional
/es/ministerios
/en/ministries
